# Model Routing

## 基本方針
SORAのスペック (GTX 1070 Ti / VRAM 8GB) を考慮し、4B〜8Bサイズの量子化モデルを中心に活用します。

## 用途ごとのモデル
- **初期の DEFAULT_MODEL**: `gemma3:4b`
- **通常会話**: `gemma3:4b`
- **長め要約**: `qwen3:8b` (候補)
- **コード・エラー読解**: `qwen2.5-coder:7b` (候補)

## 将来構想
- `router_agent` を導入し、タスクの性質に応じて自動的に最適なモデルを選択する仕組みを検討しています。
